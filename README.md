Proton - легковесная библиотека для создания многопользовательских игр с возможностью изменения кода на стороне сервера для создания разных игровых режимов.

Я создал её в 2022 году, будучи в 9 классе, как хобби-проект, желая заменить Photon PUN на своё более гибкое решение.

Была использована в игре (https://play.google.com/store/apps/details?id=com.kuzheren.MysticalTree) со средним онлайном в 5 игроков, но из-за нестабильности была заменена на Photon PUN.
